# FOSS4G2021_Tutorial

2021/12/05に行われる FOSS4G Japan 2021 Online のチュートリアルセッションの資料及び実習用教材です。

* https://www.osgeo.jp/events/foss4g-2021/foss4g-2021-japan-online/foss4g-japan-2021-online-tutorial-day#tutorial1
